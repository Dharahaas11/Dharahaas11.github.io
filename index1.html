Translator:-

It converts high-level language to machine-level language.

Types of Translators:-

There are two main types of translators: 1.

Compiler

:whole source code is converted to binary-code at once. Example Languages:- C, C++, Java (bytecode). 2.

Interpreter

:converts source code into binary-code line-by-line. Example Languages:- Python, JavaScript, Ruby.
